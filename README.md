# Zapret + GUI & Optimizer

Это модифицированная версия оригинального Zapret для Windows. Я добавил удобный графический интерфейс, переписал и оптимизировал код, а также автоматизировал настройку, чтобы не приходилось всё делать руками через консоль.

### Что изменено и добавлено:

* **Графический интерфейс (GUI):** Больше не нужно запускать скрипты через командную строку. Всё управление, запуск и остановка происходят в одно окно.
* **Глубокий анализ батников:** Утилита детально проверяет каждый `.bat` файл перед стартом. Она сама валидирует конфиги и проверяет их на совместимость с системой, чтобы ничего не ломалось на полпути.
* **Автоматическое отключение мешающих служб:** Программа сама находит и гасит службы Windows (и сторонний софт), которые могут конфликтовать с Zapret или перехватывать трафик.
* **Оптимизация кода:** Почистил исходники, убрал лишний мусор. Скрипты работают быстрее, запускаются мгновенно и не нагружают систему в фоне.

### Как запустить:

1. Скачайте архив из раздела **Releases** и распакуйте в любую папку.
2. Запустите главный файл (`Run Gui.bat`) обязательно **от имени администратора** (это нужно для управления службами и сетью).
3. Выберите нужный режим и нажмите кнопку запуска.
